# DODATAK: PM2 brzi testovi i održavanje

## A) Status procesa
pm2 status

## B) Logovi aplikacije (uživo)
pm2 logs ai-shelf-life               # izađi s Ctrl+C
pm2 logs ai-shelf-life --lines 100   # zadnjih 100 linija

## C) Restart / Stop / Start
pm2 restart ai-shelf-life
pm2 stop ai-shelf-life
pm2 start ai-shelf-life

## D) Autostart nakon restarta računala
pm2 save                 # spremi trenutne procese
pm2-startup install      # omogući PM2 autostart (Windows startup task)

## E) Ako nešto “zapne” nakon restarta
pm2 resurrect            # vrati spremljene procese (nakon pm2 save)

## F) Uklanjanje iz PM2
pm2 delete ai-shelf-life
pm2 save                 # spremi promjene u autostart listu
